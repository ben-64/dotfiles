# Installation

```bash
$ git clone https://github.com/ben-64/dotfiles.git
$ cd dotfiles
$ git submodule update --init --recursive
```

# Dependencies

- fzf

```bash
$ git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
$ ~/.fzf/install
```

## Optionnal
